인터페이스
 -  인터페이스는 멤버변수, 추상메소드만 가질 수 있다. 멤버변수도 무조건 final키워드로 상수인 변수만 가질 수 있다.
 - 변수는 무조건 public static final으로 시작해야되는데 생략이 가능하다.
   모든 메소드는 public abstract로 시작해야된다. 
 - 상속(사용)방법🛠️ [ public 클래스명 implements 인터페이스명 {} ]
 - default 메소드는 인터페이스의 메소드에 {구현부}까지 만들 수 있는 메소드 + 인터페이스에서만 사용 가능
 - 클래스상속(클래스는다중상속x)과 다르게 다중 상속이 가능함.[🛠️ public 클래스명 implements 인터페이스1, 인터페이스2 {} ]
 - interface 끼리 상속 할 땐 extends 키워드를 사용. // 이 방식은 자주 사용하지 않는다.
 - 인터페이스에도 static 메소드를 만들 수 있다.
